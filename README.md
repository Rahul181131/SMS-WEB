# Student Management System

## Overview

A Full Stack Student Management System developed using Spring Boot, MySQL, HTML, CSS and JavaScript.

## Features

- User Registration
- User Login
- Role Based Access (Admin/User)
- Add Student
- Update Student
- Delete Student
- Search Student
- Dashboard
- Export Report
- Responsive UI

## Technologies

Backend
- Java
- Spring Boot
- Spring Data JPA
- MySQL

Frontend
- HTML
- CSS
- JavaScript

Tools

- GitHub
