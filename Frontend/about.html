About Student Management System

The Student Management System (SMS) is a modern web application designed to simplify student record management. It helps educational institutions manage student information efficiently through a secure and user-friendly interface.

Our Mission

Our mission is to provide a reliable, secure, and easy-to-use platform for managing student information digitally. The system reduces paperwork, improves data accuracy, and saves valuable time for administrators, faculty members, and academic staff.

Key Features

Add Students

Create and store student records quickly.

Update Records

Edit student details whenever required.

View Students

Display all student records in one place.

Delete Students

Remove inactive or duplicate student records.

Search Students

Find students using ID, name, or email.

Secure Access

Admin login with secure authentication.

Project Objectives

The primary objective of this project is to build a complete Java Full Stack application that performs CRUD (Create, Read, Update, Delete) operations through REST APIs and stores student information in a MySQL database. The application demonstrates modern web development practices, responsive design, and backend integration using Spring Boot.
